AI Lab — RAG Glassbox & Diff Explorer

Deux mini-outils pédagogiques, qui utilisent votre endpoint Vercel pour appeler un LLM. Tout est transparent et commenté.

1) RAG Glassbox vous montre, pas à pas, comment l’IA répond à partir de votre PDF : ce qu’elle lit, ce qu’elle garde, ce qu’elle cite, et si la réponse est bien “ancrée”.
2) Diff Explorer vous fait comparer une version “avant/après” générée par l’IA, puis exécute les DEUX prompts sur le même modèle pour que vous voyiez la différence réelle dans la sortie.
Route API :
Par défaut, les appels passent par votre serveur Vercel qui détient la clé OpenAI. “Custom API key” permet de tester un autre compte/modèle sans jamais exposer votre clé côté client (elle n’est pas stockée).

Document & Question

Comment ça marche ? (explication courte)
  • Extraction : on convertit le PDF en texte côté navigateur (pdf.js).
  • Chunking : le texte est découpé en passages de ~800–1000 caractères, avec recouvrement.
  • Ranking (TF-IDF) : on mesure la proximité question ↔ passages et on sélectionne les K meilleurs.
  • Prompting : on envoie au LLM la question + les passages retenus, en exigeant des citations [n].
  • Glassbox : vous voyez les chunks, les scores, la timeline détaillée, et une jauge de “grounding”.
